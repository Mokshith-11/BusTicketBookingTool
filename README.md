# Bus Ticket Booking System

## Problem Statement
Develop a centralized bus ticket booking platform where users can search buses, view schedules, book seats, and make payments.

- Bus Management Module
- Route & Trip Module
- Booking Module
- Payment Module
- Review Module
- Admin Module
